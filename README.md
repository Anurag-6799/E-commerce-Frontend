# E-Commerce Frontend Project

This is a simple E-Commerce website frontend built with HTML, CSS, and JavaScript. It features a responsive design with product carousels (Bootstrap) and various product category pages.

## Features

-   Responsive Home Page with Product Carousel
-   Product Categories: Men, Women, Kids
-   Individual Category Pages (Shirts, Pants, Dresses, etc.)
-   Cart and Login pages (Frontend only)
-   Contact Form

## Technologies Used

-   HTML5
-   CSS3
-   JavaScript
-   Bootstrap 5

## Setup

1.  Clone the repository.
2.  Open `index.html` in your browser.
